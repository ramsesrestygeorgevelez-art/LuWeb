# LuWeb
An Catweb executor for Luau. For example: 
```lua
local hi = "test"
```
turns to:
<img width="496" height="35" alt="image" src="https://github.com/user-attachments/assets/400a5416-3f33-4b57-b65d-56913f491b4c" />

You can make functions:
```lua
local Function = LUWEB.Function(function()
    print("hi")
end)
```
to:
